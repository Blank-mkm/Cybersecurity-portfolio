\# 🔐 Security+ Acronyms (SY0-701)



This document contains a structured list of commonly used acronyms for the \*\*CompTIA Security+ (SY0-701)\*\* exam, organized by domain.



\---



\## 📚 Domains Covered



\- General Security Concepts

\- Threats, Vulnerabilities, and Mitigations

\- Security Architecture

\- Security Operations

\- Security Program Management \& Oversight



\---



\## 🔑 General Security Concepts



| Acronym | Meaning | Description |

|--------|--------|------------|

| CIA | Confidentiality, Integrity, Availability | Core principles of information security |

| AAA | Authentication, Authorization, Accounting | Framework for access control |

| IAM | Identity and Access Management | Managing user identities and permissions |

| RBAC | Role-Based Access Control | Access based on user roles |

| ABAC | Attribute-Based Access Control | Access based on attributes |

| MFA | Multi-Factor Authentication | Multiple verification methods |

| SSO | Single Sign-On | One login for multiple systems |



\---



\## ⚠️ Threats, Vulnerabilities, and Mitigations



| Acronym | Meaning | Description |

|--------|--------|------------|

| XSS | Cross-Site Scripting | Injecting malicious scripts into web apps |

| SQLi | SQL Injection | Injecting malicious SQL queries |

| DoS | Denial of Service | Overloading a system to make it unavailable |

| DDoS | Distributed Denial of Service | DoS from multiple systems |

| MITM | Man-in-the-Middle | Intercepting communication |

| RCE | Remote Code Execution | Running code remotely on a system |

| CSRF | Cross-Site Request Forgery | Forcing user actions without consent |



\---



\## 🏗️ Security Architecture



| Acronym | Meaning | Description |

|--------|--------|------------|

| VPN | Virtual Private Network | Secure remote connection |

| VLAN | Virtual Local Area Network | Network segmentation |

| DMZ | Demilitarized Zone | Isolated network segment |

| IDS | Intrusion Detection System | Detects suspicious activity |

| IPS | Intrusion Prevention System | Blocks malicious activity |

| NAC | Network Access Control | Controls device access |

| ZTNA | Zero Trust Network Access | No implicit trust model |



\---



\## 🛠️ Security Operations



| Acronym | Meaning | Description |

|--------|--------|------------|

| SIEM | Security Information and Event Management | Log analysis and monitoring |

| SOC | Security Operations Center | Security monitoring team |

| EDR | Endpoint Detection and Response | Endpoint threat detection |

| NDR | Network Detection and Response | Network threat detection |

| SOAR | Security Orchestration, Automation, and Response | Automated incident response |

| IR | Incident Response | Handling security incidents |

| IOC | Indicator of Compromise | Evidence of a breach |



\---



\## 🏢 Security Program Management \& Oversight



| Acronym | Meaning | Description |

|--------|--------|------------|

| GRC | Governance, Risk, Compliance | Managing security policies and risks |

| SLA | Service Level Agreement | Service performance contract |

| SLE | Single Loss Expectancy | Expected loss from one incident |

| ALE | Annual Loss Expectancy | Yearly expected loss |

| RTO | Recovery Time Objective | Time to restore systems |

| RPO | Recovery Point Objective | Acceptable data loss |



\---



\## 🧠 Notes



\- This list focuses on \*\*high-value acronyms\*\* commonly seen in Security+ and SOC roles.

\- Understanding the \*\*context and usage\*\* is more important than memorization.

\- These acronyms are frequently used in \*\*real-world cybersecurity environments\*\*.
