# \# 🔐 Security Controls Study Notes — Security+ SY0-701

# 

# Study notes covering common security control types for the CompTIA Security+ exam.

# 

# \---

# 

# \# 🛡️ Security Control Categories

# 

# \## 🔹 Technical Controls (Logical Controls)

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Protect systems using technology |

# | Focus | Automated security enforcement |

# | Examples | Firewalls, IDS/IPS, MFA, Encryption |

# 

# \### Exam Clue

# If the question mentions:

# \- software

# \- authentication

# \- automated systems

# \- encryption

# 

# ➡️ Think \*\*Technical Control\*\*

# 

# \---

# 

# \## 🔹 Managerial Controls (Administrative Controls)

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Manage security policies and risk |

# | Focus | Governance and planning |

# | Examples | Policies, audits, compliance reviews |

# 

# \### Exam Clue

# If the question mentions:

# \- policies

# \- procedures

# \- governance

# \- compliance

# 

# ➡️ Think \*\*Managerial Control\*\*

# 

# \---

#

# 

# | Topic | Details |

# | --- | --- |

# 

# \### Exam Clue

# If the question mentions:

# \- maintenance

# \- recovery

# \- training

# \- daily procedures

# 

# ➡️ Think \*\*Operational Control\*\*

#

# 

# \## 🔹 Physical Controls

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Protect physical assets |

# | Focus | Prevent physical access |

# 

# \### Exam Clue

# \- equipment

# \- people

# 

# ➡️ Think \*\*Physical Control\*\*

# 

# \# 🚨 Functional Security Controls

# 

# \## 🛑 Preventive Controls

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Stop incidents before they occur |

# | Goal | Reduce attack success |

# | Examples | Firewalls, MFA, ACLs |

# 

# \### Exam Clue

# If the control:

# \- blocks

# \- stops

# \- prevents

# 

# ➡️ Think \*\*Preventive Control\*\*

# 

# \---

# 

# \## 🔍 Detective Controls

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Detect malicious activity |

# | Goal | Alert security teams |

# | Examples | IDS, SIEM, CCTV, log reviews |

# 

# \### Exam Clue

# If the control:

# \- monitors

# \- detects

# \- alerts

# 

# ➡️ Think \*\*Detective Control\*\*

# 

# \---

# 

# \## 🔧 Corrective Controls

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Restore systems after incidents |

# | Goal | Minimize damage |

# | Examples | Backups, reimaging, recovery plans |

# 

# \### Exam Clue

# If the control:

# \- restores

# \- repairs

# \- fixes

# 

# ➡️ Think \*\*Corrective Control\*\*

# 

# \---

#

# 

# | Topic | Details |

# | --- | --- |

# | Purpose | Discourage attackers |

# | Goal | Reduce likelihood of attacks |

# | Examples | Warning signs, guards, cameras |

# 

# \### Exam Clue

# If the control:

# \- discourages

# \- warns

# \- intimidates

# 

# ➡️ Think \*\*Deterrent Control\*\*

#

#

#
